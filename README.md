# Web Portal UI Mockups

Цей проєкт містить UI-макети (HTML/CSS/JS) для веб-порталу управління фондом гуртожитків. Оригінальний дизайн проєкту доступний за посиланням: [Figma Design](https://www.figma.com/design/RltTCHgC2QTDXNk7IRFlLd/Web-Portal-UI-Mockups).

## Вимоги до системи

Для запуску проєкту потрібно встановити середовище **Node.js**. 

---

### Крок 1
1. Розпакуйте архів із завантаженими файлами у зручну папку.
2. Відкрийте цю папку у вашому редакторі коду (наприклад, VS Code) або відкрийте термінал (командний рядок) безпосередньо в цій папці.

### Крок 2
Введіть у терміналі наступну команду та натисніть **Enter**:
```bash
npm i

### Крок 3
Запустіть сервер розробки командою:
```bash
npm run dev

### Крок 4

Коли сервер запуститься, у терміналі з'явиться локальне посилання. ПОсилання можна відкрити як в терміналі, так і в браузері.
